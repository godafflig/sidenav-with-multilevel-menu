Sokoban

Skoban est un jeu de réflexion captivant basé sur le célèbre jeu de puzzle Sokoban, conçu pour les navigateurs web grâce à l'utilisation habile de HTML, CSS, et JavaScript. Dans ce jeu, le joueur est chargé de déplacer des caisses dans un entrepôt en vue de les placer sur des emplacements ciblés, représentés par des marqueurs distincts. L'interface utilisateur de Skoban est épurée et intuitive, permettant aux joueurs de se concentrer pleinement sur les défis de puzzle qui s'intensifient à chaque niveau.
