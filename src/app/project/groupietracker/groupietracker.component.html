Deeztrack

notre site utilise une api de musique elle vous permet de vous informer sur un panel d'artiste notre site Deeztrack est un site qui centralise la donnée d'une api . il a une fonctionnalité de barre de recherche il vous suffit d'entrer un nom d'artiste present dans notre api et ensuite de cliquer sur son logo pour acceder au information de l'artiste choisie vous pouvez aussi filtrer les artistes par date de création, date du premier album et par nombre de groupes pour cela il faut cocher le filtre de gauche puis entrer les données dans les cases correspondantes
